Stock Trading Bot

Telegram-бот для прогнозирования цен акций

Описание проекта

Stock Trading Bot — это Telegram-бот для прогнозирования цен акций и генерации торговых рекомендаций на основе исторических данных.
Бот использует несколько моделей машинного обучения и автоматически выбирает лучшую из них для конкретного тикера.

Прогноз строится на 30 дней вперед, дополнительно рассчитываются торговые рекомендации и потенциальная прибыль.


Основные возможности

** Загрузка исторических данных акций с Yahoo Finance

** Автоматический выбор лучшей модели из нескольких алгоритмов

** Прогноз цен на 30 дней вперед

** Генерация торговых рекомендаций (покупка / продажа)

** Расчет потенциальной прибыли

** Визуализация прогноза

** Логирование всех пользовательских запросов и результатов

Поддерживаемые модели

Random Forest — классическая ML-модель на табличных признаках

ARIMA — статистическая модель временных рядов

Prophet — модель прогнозирования временных рядов от Facebook

LSTM — нейросетевая модель для анализа временных рядов

Бот обучает все модели и выбирает лучшую по метрике RMSE.



Быстрый старт
1. Предварительные требования

Python 3.12 или выше

Telegram-аккаунт

Токен бота, полученный у @BotFather

2. Установка
Вариант A: Использование conda
conda create -n stock_bot_env python=3.12 -y
conda activate stock_bot_env
pip install -r requirements.txt

Вариант B: Использование virtualenv
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

3. Настройка

Получите токен Telegram-бота у @BotFather

Создайте файл .env в корне проекта:

TELEGRAM_BOT_TOKEN=ваш_токен_здесь


При необходимости настройте дополнительные параметры в коде

4. Запуск бота
python bot.py


После запуска в консоли появится сообщение:

Бот запущен...

Использование
Команды бота

/start — начать диалог и ввести тикер

/help — показать справку

/cancel — отменить текущую операцию
